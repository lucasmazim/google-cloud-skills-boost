# Getting Started with Terraform for Google Cloud

## Introdução
### O que é Terraform?
- Terraform é uma ferramenta **open source de Infrastructure as Code** (IaC) criada pela **HashiCorp**.
- Usada para **provisionar recursos**, incluindo recursos do **Google Cloud**, com arquivos de configuração **declarativos**.

### Benefícios do Terraform
- Arquivos de configuração podem ser:
  - **Compartilhados** entre membros da equipe.
  - **Tratados como código**: editados, revisados e versionados.
- Permite construir e implantar infraestruturas em nuvem complexas com **comandos simples**.
